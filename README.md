# Sales-report
Este repositório contem uma analise feita para um desafio de projeto do BootCamp Power BI e Dashboards da DIO.
---
* Contém análises feitas com:
* Gráfico de barra 
* Mapa 
* TreMap
* Gráfico de cascata
* Gráfico de Donut (Donut Chart)
* Gráfico  com visual de árvore
* Adaptação para criar um dashboard gerencial para tomadas de decisão
